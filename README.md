# Cloud-Computing-and-IOT
This project implements a Smart Forest Fire Detection and Alert System using IoT and cloud computing technologies. The system utilizes an ESP32 microcontroller with smoke and flame sensors to detect fire in real time, while a GPS module provides location information. Sensor data is transmitted via MQTT to a cloud platform hosted on AWS EC2, where Node-RED processes the data and triggers AWS Lambda and SNS to send instant email alerts. The solution enables continuous monitoring, early fire detection, and rapid emergency response through automated notifications and real-time cloud connectivity.
